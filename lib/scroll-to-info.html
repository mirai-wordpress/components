Scroll to info

Muestra un elemento con el cual podemos desplazarnos hacia otra sección de esa misma página con un efecto animado progresivo, de este modo evitamos el salto drástico que haríamos con un simple anchor.

Este ejemplo es para mostrar una flecha como elemento con el que interactuar, pero se le puede aplicar del mismo modo a un texto.


Marcado HTML

Flecha en color negro
<a href="#intro"  class="scroll-to-info"></a>
Flecha en color blanco
<a href="#intro"  class="scroll-to-info arrow-white"></a>

CSS

/* TRANSITIONS */
.scroll-to-info {
    -webkit-transition: all 0.5s ease;
    -moz-transition: all 0.5s ease;
    -ms-transition: all 0.5s ease;
    -o-transition: all 0.5s ease;
    transition: all 0.5s ease;
}

/* GENERAL */
.scroll-to-info {
    background: url("/files/arrow_down.png") no-repeat 0 0 transparent;
    bottom: 10px;
    display: block;
    height: 50px;
    left: 50%;
    margin-left: -25px;
    opacity: 0.7;
    position: absolute;
    width: 50px;
}

.scroll-to-info:hover {
    bottom: 0px;
}

.scroll-to-info.arrow-white {
    background-position: 0 -50px;
}

Javascript necesario

/* Smooth scrolling for nav sections */
Mirai.query('.scroll-to-info').click(function(evt){
  evt.preventDefault();
  var link = Mirai.query(this).attr('href');
  var pos = Mirai.query(link).offset().top;
  Mirai.query('html,body').animate({scrollTop:pos},700);
});

Modificar el punto de destino:

Cuando ponemos el típico menú anchor para facilitar la navegación de una página al bajar a la sección en cuestión el motor se posiciona fixed y perdemos el Título de dicha sección, para corregirlo lo único que tenemos que modificar es el valor del scrollTop en el JS anterior, por ejemplo:

Mirai.query('html,body').animate({scrollTop:pos-100},800);
