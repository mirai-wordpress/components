CSS Style Guide

Hemos creado la base de una guía de estilos CSS, la iremos ampliando según vaya siendo necesario.

REGLAS GENERALES

Salto de línea para todas las declaraciones, haya una sola o varias:

body {
    font-family: Open Sans;
}


Espacio entre la regla y la apertura de las declaraciones:

body {
    font-family: Open Sans;
}


Espacio de salto de línea entre reglas:
 
body {
    font-family: Open Sans;
}

content {
    padding: 0;
}


Al contextualizar varias clases juntas, éstas se escribirán seguidas sin dar salto de línea entre ellas y con un espacio entre medias:

body, footer, #main container {
    font-family: Open Sans;
}


Todas las declaraciones irán ordenadas alfabéticamente:

body {
    border-color: #000;
font-family: Open Sans;
margin: 10px 0;
}


Todas las declaraciones irán tabuladas:

body {
    border-color: #000;
margin: 10px 0;
}



Para las clases que utilicen comillas en su propia regla, se respetarán esas comillas:

.engine_mashup > div form .button-group input[type="submit"] {}


Las clases deber llevar siempre guión medio, nunca guión bajo:

class="bg-color"    ---    .bg-color


Utilizar sólo minúsculas, nunca mayúsculas.



Utilizar clases e ID's siempre en inglés.



Backgrounds por defecto en hexadecimal (si es posible abreviado), en caso de llevar transparencia RGBA.



Para las clases que determinen background utilizar "bg-" como base y el nombre del color en relación a la intensidad del mismo:

bg-light-grey   /   bg-grey   /   bg-dark-grey


Para definir margin y padding comunes utilizaremos clases genéricas:

margin: mrg - seguido de un número consecutivo: mrg1
padding: pdd - seguido de un número consecutivo: pdd1
