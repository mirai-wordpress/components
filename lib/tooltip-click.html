Ocultar Tooltip en click

Si se indica data-trigger="click" en el Tooltip de Bootstrap y quieres que al hacer click fuera de ese elemento se oculte el Tooltip, necesitarás lo siguiente:

  • HAY UN PAR DE BUGS CON ESTE CONTENIDO

Marcado HTML del ejemplo

<ul class="list-rooms"/>
    <li class="desk"/>
	<a href="#" class="mi_tooltip"  rel="tooltip" data-title="Escritorio" data-trigger="click" data-placement="bottom" data-delay="{show: 500, hide: 100}"/>
	    <span></span>
	</a>
    </li>
    <li class="wifi"/>
	<a href="#" class="mi_tooltip"  rel="tooltip" data-title="Wifi Gratis" data-trigger="click" data-delay="{show: 500, hide: 100}"/>
	    <span></span>
	</a>
    </li>
    <li class="phone"/>
	<a href="#" class="mi_tooltip"  rel="tooltip" data-title="Teléfono" data-trigger="click" data-placement="bottom" data-delay="{show: 500, hide: 100}"/>
	    <span></span>
	</a>
    </li>
    <li class="wakeup"/>
	<a href="#" class="mi_tooltip"  rel="tooltip" data-title="Servicio despertador" data-trigger="focus" data-placement="bottom" data-delay="{show: 500, hide: 100}"/>
	    <span></span>
	</a>
    </li>
</ul/>

Javascript necesario

jQuery(document).ready(function() {
     jQuery('.mi_tooltip').tooltip();
     jQuery(".mi_tooltip").click(function(event){
       event.preventDefault();
     jQuery('body').on('click', function (e) {
         jQuery('.mi_tooltip').each(function () {
             //the 'is' for buttons that trigger popups
             //the 'has' for icons within a button that triggers a popup
             if (!jQuery(this).is(e.target) && jQuery(this).has(e.target).length === 0 && jQuery('.erase-tooltip').has(e.target).length === 0) {
                 jQuery(this).tooltip('hide');
             }
         });
     });
 });
ANOTACIÓN: El preventDefault es necesario cuando el data-trigger es "click".
