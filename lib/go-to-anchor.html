Go To Anchor

Se le aplica una transición a un link con anchor.


Marcado HTML

Sólo haría falta ponerle la clase "go-to-anchor", ejemplo:
<span class="go-to-anchor"></span>

Javascript necesario

/* GoToAnchor all pages */
Mirai.Dom.goToAnchor();

Modificar el punto de destino:

Para poder salvar el espacio que ocupa el Motor o el Header en Affix se puede modificar valor de la posición directamente en el JS anterior, basta con añadir el valor deseado entre comillas simples:

/* GoToAnchor all pages */
Mirai.Dom.goToAnchor('150');

NOTA: El valor por defecto es 100.

Aplicarlo sólo a ciertas páginas:

Con este JS podemos aplicar esta transición sólo a algunas páginas, con esto evitamos que el Script se ejecute siempre.
Para aplicarlo a determinadas páginas habría que hacerlo por la clase "cat-" que se asigna a cada una de ellas a través del Audit. Ejemplo para aplicarlo a 2 páginas:

/* GoToAnchor some pages */
Mirai.query(document).ready(){
    if (Mirai.query('BODY').hasClass('cat-location') || Mirai.query('BODY').hasClass( 'cat-rooms'))  {
        Mirai.Dom.goToAnchor('200');
    }
}
